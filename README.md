## What Is This?

A `.cursorrules` file lives in your project root and gives [Cursor AI](https://cursor.sh) persistent context about your stack, conventions, and code style. Every time Cursor generates code, it reads these rules automatically.

**Without rules:** Cursor generates generic boilerplate that fights your conventions.  
**With these rules:** Cursor writes code that looks like a senior dev already on your team wrote it.
